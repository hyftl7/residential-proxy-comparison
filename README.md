## 住宅代理是什么：一句话版本

住宅代理（Residential Proxy），指的是由真实用户设备发出请求的 IP 地址——具体就是普通居民家里的宽带网络。目标网站看到的流量，跟隔壁邻居打开 Chrome 浏览网页没有区别。

这和机房代理（Datacenter Proxy）有本质不同。机房代理 IP 来自云服务器或数据中心，而住宅代理 IP 来自 AT&T、Comcast 这类真实宽带运营商。网站的风控系统对两者的信任程度完全不同。

---

## 住宅代理的两种主要形态：静态 vs 动态

这是做住宅代理对比时最容易被忽略的分叉口。

**动态轮换住宅代理**：你购买的是流量包，每次请求可能走不同的 IP，IP 来自全球各地真实用户设备构成的代理池。适合大规模数据采集、SEO 监测、广告核查这类对 IP 稳定性要求不高、但需要频繁切换地址的场景。按 GB 计费，池子越大越好。

**静态住宅代理（家宽 IP）**：你持有一个固定 IP，这个 IP 真实归属于某个住宅宽带账户。长期不变，独享使用，不与其他人共享。适合社媒账号长期养号、跨境电商账号管理、直播运营这类需要"一个设备、一个身份"长期稳定的场景。

这两类产品经常被混为一谈，但适用场景几乎不重叠。

A：TikTok 账号运营强烈推荐静态家宽。动态轮换意味着每次登录的 IP 可能不同，平台会识别出异常行为，风控概率更高。静态家宽 IP 保持固定，模拟的是"一台设备固定在美国某个地址上网"的状态，更符合平台对真实用户的预期。

**Q：住宅代理和 ISP 代理有什么区别？**

A：ISP 代理（也叫静态住宅代理）通常是服务商直接向运营商购买 IP 段，在数据中心部署但归属显示为 ISP，速度快但有些检测工具能识别出数据中心特征。真实家宽 IP 则是实际接入住宅宽带网络，检测工具无法区分和真实用户的差异。VIRCS 属于后者。
